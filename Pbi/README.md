# Zabbix-Template
файлы шаблонов ZABBIX v6.2.6 для ресиверов PBI DMM
В шаблонах представлены следцующие элементы данных собираемые по SNMP OID;
* C/N (dB)
* Eb/N0 (dB)
* Strength (dBm)
* Satellite Frequency (MHz)
*	Symbol Rate (KBaud)
*	Tuner Status
*	Decoder 1 Service Type
*	Decoder 2 Service Type
